# 디자인 카테고리 가이드라인

## 1. Landing Page (랜딩 페이지)
**목적**: 제품/서비스를 소개하고 방문자를 고객으로 전환

**필수 요소**:
- Hero Section: 강렬한 첫 인상, 핵심 가치 제안 (Value Proposition)
- CTA Buttons: 명확한 행동 유도 버튼 (Get Started, Sign Up 등)
- Features/Benefits: 3-6개의 주요 기능이나 장점
- Social Proof: 고객 후기, 통계, 로고 등
- Footer: 연락처, 링크

**디자인 특징**:
- 시각적 임팩트 (큰 타이포그래피, 그라디언트)
- 단순하고 직관적인 메시지
- 스크롤 기반 스토리텔링
- 반응형 디자인

**레이아웃 종류**:
1. Hero-Centered: 중앙 정렬, 대형 타이틀
2. Split-Screen: 좌우 분할 (텍스트 + 이미지/폼)
3. Video Background: 동영상 배경
4. Asymmetric: 비대칭 레이아웃
5. Minimal: 미니멀 디자인

---

## 2. Dashboard (대시보드)
**목적**: 데이터와 정보를 시각적으로 표시하고 관리

**필수 요소**:
- Navigation: 사이드바 또는 상단 네비게이션
- Stats Cards: KPI, 통계 카드 (매출, 사용자, 주문 등)
- Charts/Graphs: 데이터 시각화 (라인, 바, 도넛 차트)
- Tables: 데이터 테이블 (최근 주문, 사용자 목록)
- Activity Feed: 최근 활동 피드
- Filters/Search: 필터, 검색 기능

**디자인 특징**:
- 정보 계층 구조 명확
- 스캔 가능성 (Scanability)
- 데이터 밀도 균형
- 색상으로 상태 표시 (성공: 녹색, 경고: 노란색, 위험: 빨간색)

**레이아웃 종류**:
1. Sidebar Layout: 왼쪽 사이드바 + 메인 콘텐츠
2. Top Nav: 상단 네비게이션 + 탭
3. Card-Based: 카드 그리드 레이아웃

---

## 3. E-commerce (이커머스)
**목적**: 제품을 판매하고 구매 프로세스를 관리

**필수 요소**:
- Product Grid: 제품 그리드 (썸네일, 이름, 가격)
- Product Cards: 제품 카드 (이미지, 가격, 평점, 카트 버튼)
- Filter/Sort: 필터, 정렬 기능
- Shopping Cart Icon: 장바구니 아이콘 (개수 표시)
- Product Detail: 제품 상세 페이지 (갤러리, 가격, 설명, 리뷰)
- Add to Cart Button: 장바구니 담기 버튼

**디자인 특징**:
- 제품 이미지 중심
- 명확한 가격 표시
- 평점/리뷰 시각화
- 신뢰성 (보안, 배송 정보)

**레이아웃 종류**:
1. Product Grid: 3-4열 제품 그리드
2. Product Detail: 이미지 갤러리 + 구매 정보
3. Checkout: 장바구니/결제 페이지

---

## 4. Portfolio (포트폴리오)
**목적**: 작업물, 프로젝트를 시각적으로 전시

**필수 요소**:
- Project Gallery: 프로젝트 갤러리
- Project Cards: 프로젝트 카드 (썸네일, 제목, 카테고리)
- Project Detail: 프로젝트 상세 (이미지, 설명, 기술 스택)
- About Section: 자기소개, 경력
- Contact: 연락 방법

**디자인 특징**:
- 시각적 중심 (큰 이미지)
- 여백 활용
- 미니멀한 텍스트
- 개성 있는 레이아웃

**레이아웃 종류**:
1. Masonry Grid: 벽돌 쌓기 레이아웃 (다양한 높이)
2. Minimal Fullscreen: 전체 화면 프로젝트 슬라이드
3. Grid with Hover: 호버 시 정보 표시

---

## 5. Blog (블로그)
**목적**: 글을 읽고 정보를 얻는 플랫폼

**필수 요소**:
- Post List: 글 목록
- Post Card: 글 카드 (썸네일, 제목, 요약, 날짜, 작성자)
- Post Header: 글 헤더 (제목, 날짜, 작성자, 카테고리)
- Post Content: 본문 (Typography 중심)
- Sidebar: 사이드바 (카테고리, 인기 글, 태그)
- Comments: 댓글 섹션

**디자인 특징**:
- 가독성 최우선 (폰트, 줄 간격, 여백)
- Typography 중심
- 콘텐츠 계층 구조
- 이미지와 텍스트 균형

**레이아웃 종류**:
1. Grid Layout: 2-3열 글 그리드
2. Magazine Style: 매거진 스타일 (대형 featured post)
3. List View: 리스트 뷰 (이미지 + 텍스트)

---

## 6. Components (컴포넌트 라이브러리)
**목적**: 재사용 가능한 UI 요소를 전시

**필수 요소**:
- Button Variants: 버튼 변형 (Primary, Secondary, Outline, Icon)
- Form Elements: 입력 필드 (Input, Textarea, Select, Checkbox, Radio)
- Cards: 카드 변형 (기본, 이미지, 통계)
- Navigation: 네비게이션 (Navbar, Tabs, Breadcrumb)
- Alerts/Notifications: 알림, 경고
- Modals/Dialogs: 모달, 다이얼로그

**디자인 특징**:
- 일관성 (색상, 간격, 크기)
- 상태 표시 (hover, active, disabled)
- 접근성 고려
- 명확한 라벨링

**레이아웃 종류**:
1. Showcase Grid: 컴포넌트 그리드 전시
2. Storybook Style: 각 컴포넌트별 섹션
3. Interactive Demo: 인터랙티브 데모

---

## 색상 사용 가이드

각 카테고리에서 색상 사용:
- **Primary Color**: 주요 버튼, 링크, 강조 요소
- **Secondary Color**: 보조 버튼, 배경 그라디언트
- **Accent Color**: 아이콘, 배지, 하이라이트

## 타이포그래피 가이드

- **Headings**: 굵고 큰 폰트 (64-96px)
- **Body Text**: 읽기 쉬운 크기 (16-20px)
- **Labels**: 작고 명확한 텍스트 (12-14px)
